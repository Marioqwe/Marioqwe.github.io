HOW TO PLAY: Complete each level by making all of the tile's values equal to the current level. The first tile you select will increase by one. The second tile you select will increase by the first one's value. Keep in mind that the tile's values cannot be greater than the current level. Example: select tile with value 5 then select tile with value 8. First tile is 5 and second tile is 8. So 5+1=6 and 8+5=13. Enjoy!
