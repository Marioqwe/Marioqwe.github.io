HOW TO PLAY: Select TWO tiles to merge them. The FIRST tile you select will INCREASE BY ONE. The SECOND tile you select will INCREASE BY THE VALUE OF THE FIRST ONE. Example: select tile with value 5 then select tile with value 8. First tile is 5 and second tile is 8. So 5+1=6 and 8+5=13. Enjoy!
